== Description ==

This plugin allows customers to autocomplete billing and shipping addresses on the checkout page using the NZpost Parcel Address API.
The ParcelAddress API enables search for international and domestic addresses. This API can be used for auto-complete (or type-ahead) dialogues for faster addressing entry. It is also used to validate addresses.When the customer enter on addresses field, then the corresponding Address with street,city of Newzeland automatically populate with the selected address data.

In order for you to gain access to the APIs, you and your organisation must meet the following pre-requisites:

1.Your organisation must have an active NZ Post or CourierPost account number.
2.You must be registered with the Developer Portal
3.You must have reviewed and accepted the Terms and Conditions for use of the NZ Post Group APIs.
4.You must have requested and received access to the API.

You can register <a href="https://login.nzpost.co.nz/login" target="_blank" title="login to login.nzpost.co.nz">here</a> by completing the registration form


= Features: =

* Parcel Address Autocomplete Suggestions
* Address suggestions are relative to the selected country
* Supports the latest version of Wordpress

== Installation ==

To install Parcel Address Checkout Autocomplete, follow these steps:

1. Download and unzip the plugin
2. Upload the entire parcel-address-autocomplete-woocommerce/ directory to the /wp-content/plugins/ directory
3. Activate the plugin through the Plugins menu in WordPress
